# 参数调整功能 - 快速启动指南

## 🚀 5 分钟上手

### 步骤 1: 启动服务

```bash
# 后端
python -m uvicorn app.main:app --host 0.0.0.0 --port 8080

# 前端（新终端）
cd frontend
npm install
npm run dev
```

### 步骤 2: 打开前端

访问 `http://localhost:3000`

### 步骤 3: 配置设置

1. 点击右上角 **⚙️ 设置** 按钮
2. 填写 LLM 配置（如果需要智能情感分析）
3. 调整 TTS 参数

### 步骤 4: 测试参数

#### 测试 1: 调整语速
```
1. 设置 → 语速滑块 → 拖到 1.5x
2. 保存并应用
3. 发送消息："这是快速语速测试"
4. 听听效果
```

#### 测试 2: 调整情感
```
1. 设置 → 情感 → 选择 "happy"
2. 保存并应用
3. 发送消息："太棒了！这真是太令人兴奋了！"
4. 听听效果
```

#### 测试 3: 调整高级参数
```
1. 设置 → 展开"高级参数"
2. Temperature → 拖到 0.5（更稳定）
3. 保存并应用
4. 发送消息："欢迎收听今天的新闻播报"
5. 听听效果
```

## 💡 小贴士

1. **从默认开始**: 先用默认参数，确认功能正常
2. **逐步调整**: 一次只改一个参数，观察效果
3. **保存配置**: 找到满意的配置后记录下来
4. **恢复默认**: 不满意时点击"恢复默认"

## 🐛 遇到问题？

### 问题 1: 参数不生效
- 确认点击了"保存并应用"
- 刷新页面重试

### 问题 2: 音色不存在
- 检查 `presets/` 目录
- 使用 `default` 音色

### 问题 3: 前端无法连接
- 确认后端已启动（端口 8080）
- 检查 TTS API URL 设置
