# 角色文件夹 (char)

此文件夹用于存放自定义角色配置。每个子文件夹代表一个角色。

## 文件夹结构

```
char/
├── 小明/
│   ├── voice.wav       # 角色专属音色文件 (必需)
│   └── config.json     # 角色配置文件 (可选)
├── 小红/
│   ├── default.wav
│   └── config.json
└── ...
```

## config.json 格式

```json
{
  "system_prompt": "你是小明，一个开朗活泼的男孩，说话风格轻松幽默。"
}
```

## 说明

1. **文件夹名称**: 文件夹名称将作为角色名称显示在前端
2. **音色文件**: 需要放置一个 .wav 格式的音频文件作为角色的专属音色
3. **配置文件**: config.json 是可选的，用于设置角色的专属系统提示词
4. **系统提示词**: system_prompt 将会附加到默认系统提示词后面，用于定制角色的性格和说话风格
